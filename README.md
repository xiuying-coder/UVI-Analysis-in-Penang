# UVI-Analysis-in-Penang
This repository contains the analysis scripts used to investigate the
relationships between land surface temperature (LST) and environmental
parameters, including UVI, albedo, NDVI, and NDBI, in Penang under clear-sky
conditions.
## Contents
- `lst_environmental_correlation.R`: R script for linear correlation analysis,
  statistical testing, and figure generation.
## Data availability
The remote sensing datasets used in this study are not included in this
repository due to data size and licensing constraints.
## Purpose
This repository is intended to support methodological transparency and
reproducibility of the associated research.
